iPazar App

Welcome to the iPazar App

This app is thoughtfully designed to offer in-depth insights into the Istanbul Stock Exchange and global markets.

Features

Features

Our Application provides the following features:

Live Prices

iPazar is a Data Feed Vendor for the Istanbul Stock Exchange.
In our iPazar Application, you can monitor real-time movements of Istanbul Stock Exchange Stocks, Currencies, Derivatives, Commodities, and Precious Metals.

Alerts

The iPazar Applications can send push notifications for any customizable event.
You can customize a notification to alert you in your devices for events like is the BIST 100 Index over 11,100 or did a Company reached a monthly maximum in volume.
Be the first to know by using iPazar.

Charting

iPazar Application provides informative Charting options that provide useful insights. You can customize your charts with various indicators, time frames and drawing tools.
Some of the indicators include Fibonacci Arcs, Trend Lines, Rectangles and Volume indicators.
For more infromation about charting you can watch our tutorial videos.

Calendar

Detailed calendar for upcoming corporate actions. It includes Split, Merges, Payment and all the announcements from the C board published in KAP. You can browse this Calendar and be informed for all the upcoming corporate actions.

News Updates

Get the latest economic news from Turkiye's top news vendors.
Dunya News, Milliyet, Hürriyet, CNN Turkiye and other News sources are intergrated into the iPazar Application providing economic news.
If the article talks about a specific company we will match them with the symbol and the current change.
By doing this, the user can match the article's infromation with the current change.
The user can also click on the headline to read the whole article in our App or visit the external website.

By selecting the "only actions" Button on the top left you can only see the Corporate Actions from KAP.

Data Feed

iPazar Terminal offers Live Data Feed for categories below.
Select the cetogories by clicking on the Button to see the Data Feed window.

Stocks
Derivatives
Commodities
Foreign Currencies
World Indices
